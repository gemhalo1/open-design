# Document Structures

- Report: title, summary, context, findings, recommendation, next steps.
- Resume: header, summary, experience, selected work, skills, education.
- One-pager: headline, problem, solution, proof, offer, CTA.
- Proposal: brief, objectives, scope, timeline, investment, acceptance.
- Docs page: overview, prerequisites, steps, examples, troubleshooting.

Default to one page unless the user asks for a longer document.
