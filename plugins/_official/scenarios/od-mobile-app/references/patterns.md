# Mobile Patterns

- Onboarding: welcome, value proof, permission, account setup.
- Core workflow: list/feed, detail, action sheet, confirmation.
- Commerce: browse, product detail, cart, checkout, receipt.
- Productivity: home dashboard, task list, create/edit sheet, activity.
- Settings: profile, preferences, notifications, privacy.

Prefer 4-6 screens. Include mobile states, not only happy-path screens.
