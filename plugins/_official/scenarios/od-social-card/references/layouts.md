# Social Card Layouts

- X single card: 16:9, one statement, tiny attribution, one accent.
- X quote card: oversized quote mark, pull quote, source, footer.
- LinkedIn data card: headline, one chart/proof element, interpretation.
- Instagram story: 9:16, hook, image/product zone, CTA/swipe cue.
- Rednote carousel: 3:4 or 4:5, cover, 3 value cards, checklist.
- WeChat cover pair: 21:9 header and 1:1 share card using same title system.
- Multi-platform set: preserve core message but adapt ratio and density per frame.

Frame count defaults: post/quote/data = 1, carousel = 4, WeChat = 2, multi-platform = 3-5.
