# Wireframe Layouts

- Entry screen: title, context strip, primary CTA, secondary link.
- Form/task screen: left input zone, right guidance zone, validation row.
- Dashboard screen: header, filter bar, metric row, list/table zone.
- Detail screen: summary panel, tabs, timeline/list, action rail.
- Empty/error state: icon placeholder, explanation, recovery action.

Use 4-6 screens by default. Each screen should explain one user decision.
